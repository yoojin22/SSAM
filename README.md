## 🗨️ About SSAM

SSAM은 추락하는 교권을 보호하기 위해 개발된 상담 시스템입니다.<br>
끊임없는 학부모의 연락과 폭언으로부터 교사들을 보호하기 위해 기획된 이 서비스는 교사의 업무를 지원하고 스트레스를 줄이기 위한 다양한 기능을 제공합니다. SSAM의 주요 기능은 다음과 같습니다:<br>

1. 학급 및 학생 관리
2. 화상 상담과 실시간 욕설 필터링
3. 학급 공지 챗봇 관리
